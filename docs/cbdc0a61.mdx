# **变更日志**

Tiptap 由 50 多个独立的软件包组成。以下是您遵循更改所需的一切：

1. 检查[<u>GitHub 发布页面</u>](https://github.com/ueberdosis/tiptap/releases)以获得最佳概述。
2. 有关更多详细信息，请访问[<u>GitHub 上的 Tiptaps CHANGELOG</u>](https://github.com/ueberdosis/tiptap/blob/main/CHANGELOG.md)。
3. 此外，[所有包也都有<u>一个单独的**`CHANGELOG.md`**</u>](http://所有包也都有一个单独的CHANGELOG.md)。
4. 关注[<u>GitHub 上合并的 Pull 请求</u>](https://github.com/ueberdosis/tiptap/pulls?q=is%3Apr+is%3Aclosed)也很有帮助。

[<u>我们将撰写</u>](https://tiptap.dev/blog)有关更大更改的博客文章，并将它们保存在此处的更改日志中。
